# FlappyClone_Unity
Unity로 Flappy Bird의 클론(Clone) 게임 제작

## 설명
- 화면을 터치하면 새가 조금씩 날아오름
- 가만히 있으면 새가 내려감
- 새가 바닥, 천장, 파이프에 닿으면 게임 오버
- 최대한 많은 점수를 획득하는 것이 목표
